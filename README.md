# Weather Tracker Service Application

---

### The service provides an API
#### `weatherstackAPI`
* To search current weather report by city name.

## Technologies

---
- Java 17
- Spring Boot 3.0
- Open API Documentation
- Spring Data JPA
- H2 In Memory Database
- Restful API
- Maven
- Junit5
- Mockito
- Integration Tests
- Docker
- Docker Compose
- Github Actions
- Prometheus
- Grafana
